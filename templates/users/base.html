Данное приложение было создано в учебных целях с помощью фреймворка Flask
  1. Изменяющая строка запроса в зависимости от номера курса
  2. Список добавленных пользователей в json-чик
  3. Информация о конкретном пользователе в зависимости от его id в json-чике
  4. Добавление нового пользователя в json-чик
  5. Редактирование имени пользователя в зависимости от его id, который у него указан в json-чике
  6. В адресной строке смени цифру между users/1/delete на id пользователя и ты удалишь его из json-чика.
    Посмотреть id пользователя можно перейдя по ссылке в пункте 2
    Удаление пользователя из json-чика по его id
  7. Залогинить пользователя через session
  8. Разлогинить пользователя. Удалить из session
